🌀 Phase Portrait
ė(t) vs e(t)

The phase portrait shows the trajectory of the system in the error-derivative plane. Stable systems spiral toward the origin (0, 0). The shape reveals damping characteristics:

  • Spiral inward: Stable, underdamped
  • Direct approach: Overdamped
  • Limit cycle: Marginally stable
  • Spiral outward: Unstable

📚 Complete PID Mathematical Framework

Continuous-Time PID Controller

1
u(t) = Kpe(t) + Ki∫₀ᵗ e(τ)dτ + Kdde(t)/dt
The parallel (ideal) PID form - most common representation
2
u(t) = Kp[e(t) + 1/Ti∫₀ᵗ e(τ)dτ + Tdde(t)/dt]
Standard (ISA) form where Ti = Kp/Ki and Td = Kd/Kp
3
u(t) = Kc[(1 + 1/Tis + Tds)e(t)]
Series (classical) form - historically used in analog controllers

Closed-Loop System

4
E(s) = R(s) - Y(s)
Error in Laplace domain
5
Y(s) = G(s)·U(s) = G(s)·C(s)·E(s)
Plant output equals plant transfer function times control
6
Y(s) = C(s)G(s)/1 + C(s)G(s) · R(s)
Closed-loop transfer function derivation

Discrete-Time Implementation

1
u[k] = Kpe[k] + KiTsΣⱼ₌₀ᵏ e[j] + Kd(e[k]-e[k-1])/Ts
Position form - direct implementation of PID equation with sampling period Ts
2
Δu[k] = u[k] - u[k-1]
Velocity (incremental) form - outputs change in control action
3
Δu[k] = Kp(e[k]-e[k-1]) + KiTse[k] + Kd/Ts(e[k]-2e[k-1]+e[k-2])
Expanded velocity form - preferred in practice (bumpless transfer, anti-windup)

Z-Transform

4
C(z) = Kp + KiTs/(1-z⁻¹) + Kd(1-z⁻¹)/Ts
Discrete PID transfer function using backward difference
5
C(z) = q₀ + q₁z⁻¹ + q₂z⁻²/1 - z⁻¹
Where q₀ = Kp + KiTs/2 + Kd/Ts, etc.

Laplace Transform Analysis

1
ℒ{e(t)} = E(s), ℒ{∫e(τ)dτ} = E(s)/s, ℒ{de/dt} = sE(s)
Key Laplace transform pairs
2
C(s) = U(s)/E(s) = Kp + Ki/s + Kds
PID transfer function
3
C(s) = Kds² + Kps + Ki/s
Combined over common denominator - shows one pole at origin, two zeros

First-Order Process with PID

4
G(s) = K/τs + 1
First-order process model
5
L(s) = C(s)G(s) = K(Kds² + Kps + Ki)/s(τs + 1)
Open-loop transfer function (Type 1 system)
6
T(s) = K(Kds² + Kps + Ki)/(τ + KKd)s² + (1 + KKp)s + KKi
Closed-loop transfer function

Stability Criteria

1
BIBO Stability: All poles must have Re{p} < 0
Bounded-Input Bounded-Output stability requirement
2
Routh-Hurwitz Criterion: All first column elements > 0
Necessary and sufficient for polynomial stability

Gain and Phase Margins

3
GM = 1/|L(jωφ)| where ∠L(jωφ) = -180°
Gain Margin at phase crossover frequency
4
PM = 180° + ∠L(jωc) where |L(jωc)| = 1
Phase Margin at gain crossover frequency

Nyquist Criterion

5
Z = N + P
Z = closed-loop RHP poles, N = encirclements of -1, P = open-loop RHP poles
6
Stable ⟺ Z = 0 ⟺ N = -P
For stability, Nyquist plot must encircle -1 exactly P times counter-clockwise
